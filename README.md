# 🌾 PriceHarvest

PriceHarvest is an agricultural crop price analysis platform developed as a B.Tech Data Science mini project. It analyzes historical crop price data from Agmarknet and presents insights through dashboards, Power BI visualizations, an AI chatbot, and multilingual support.

## Features
- Crop Price Analysis
- Data Visualization Dashboard
- Power BI Reports
- AI Chatbot
- Multilingual Support

## Tech Stack
- Python
- Pandas
- NumPy
- Streamlit
- HTML, CSS, JavaScript
- Power BI

## Dataset
Government of India Agmarknet Portal
